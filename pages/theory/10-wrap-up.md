# Підсумок

<v-clicks>

- Agentic IDE = **контекст + інструменти + памʼять**
- Правила — **policy**, команди — **шаблони (templates)**, навички (skills) — **workflows**, памʼять — **стан**

# Summary

<v-clicks>

- Agentic IDE = **context + tools + memory**
- Rules are **policy**, commands are **templates**, skills are **workflows**, memory is **state**
- Biggest quality boost: **Plan → Act → Verify → Update**
- Standardize “unwritten rules” → store them in the repo.

# Quick start checklist (що зробити першими 3 кроками)

<v-clicks>

1. Додайте **project rules** (коротко: стек, стиль, NEVER/ASK FIRST)
2. Оформіть 3–5 **commands** для типових задач (review/tests/docs)
3. Заведіть мінімальний **Memory Bank** (наприклад `docs/memory/activeContext.md` + `docs/memory/progress.md`)
4. Заведіть `SKILLS.md` (каталог повторюваних процедур для команди).

</v-clicks>

</template>
<template #en>

# Quick start checklist (first 3 steps)

<v-clicks>

1. Add **project rules** (brief: stack, style, NEVER/ASK FIRST)
2. Create 3–5 **commands** for common tasks (review/tests/docs)
3. Set up a minimal **Memory Bank** (e.g. `docs/memory/activeContext.md` + `docs/memory/progress.md`)
4. Add `SKILLS.md` (catalog of repeatable procedures for the team).

</v-clicks>
