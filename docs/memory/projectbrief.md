# Project Brief

## What this is

Slidev presentation for the workshop:
**“Правила, команди та навички для Agentic IDE”**.

## Primary goal

Teach a repeatable, engineering-grade approach to working with agentic tools:
**rules → commands → skills → memory → verification**.

## Audience

Developers and tech leads who already write code, but want consistent workflows with AI agents.

## Non-goals

- Not a full tutorial on prompt engineering theory.
- Not a vendor marketing deck.
